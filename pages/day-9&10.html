Day-10 of the Challenge

Summary

HTML

HTML defines our page content, structure and meaning. Learned the following till date.

  • Structure of HTML document like Head, Body, Header, Main, Section, Footer etc.
  • Inline Element which fits into a line with other elements like a, img, span etc. Although you can change it in CSS.
  • Block Elements which reserves the whole width of the screen for themselves like header, main, footer, ul, ol, li, p etc. Although you can also change it in CSS.

CSS

CSS is used for styling the page content. Learned the following properties till date.

  • text-align for aligning the content to center.
  • color to change the color like color for image or text etc.
  • backgroud-color to change the background color.
  • Margin and Padding to change the spacing.
  • border and its various properties like width, height, color, radius, style etc.
  • Selectors, Combinators, Pseudo Selectors, Classes and ID.
